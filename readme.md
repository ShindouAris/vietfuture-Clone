Vietfuture clone

Cách để vận hành project này:

1. Clone project về máy
1.1 Đảm bảo bạn đã cài đặt NodeJS và npm
2. Mở terminal và điều hướng đến thư mục project
3. Chạy lệnh `npm install` để cài đặt các dependencies
3.1 Chạy `cd vietfutureclone` nếu bạn đang ở thư mục gốc của project
4. Chạy lệnh `npm run dev` để khởi động server phát triển
5. Mở trình duyệt và truy cập `http://localhost:5173` để xem ứng dụng
